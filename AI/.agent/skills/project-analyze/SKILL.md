# Software Development

Single idea can be break it down to backend, frontend, database, QA, marketing, analysis. Write it down to single PRD file(prd-*.md) and put each domain a section.
Each domain has it dedicated agent to run it. This agent will break it down domain tasks and sent to those agents to process

- `backend`, `backend-engineer`
- `frontend`, `frontend-engineer`
- `qa`, `qa-engineer`
- `database`, `database-engineer`
- `marketing`, `marketing`
- `analysis`, `analyzer`
