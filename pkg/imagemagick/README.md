# imagemagick

### compress image
`./compress_img.sh -s <size> images...`


### identify
show details of image

`identify [-verbose] <image>`


### convert
`covert <image_from> [options] <image_to>`

options:  
- `-resize <percentage>`
- `-resize <width>`
- `-resize <height>`
- `-resize <width>x<height>`
- `-quality <value>` , compress level
